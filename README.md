# SIPRU Frontend

Frontend web untuk **Sistem Peminjaman Ruangan Kampus (SIPRU)**.  
Aplikasi ini digunakan untuk mengelola data peminjaman ruangan melalui antarmuka berbasis web yang terhubung dengan SIPRU Backend API.

## Description

SIPRU Frontend merupakan aplikasi client-side yang dibangun menggunakan React dan TypeScript.  
Aplikasi ini menyediakan fitur pengelolaan data peminjaman ruangan secara interaktif, termasuk CRUD data, update status, serta pencarian dan filter data.

## Features (Current)

- CRUD data peminjaman ruangan
- Update status peminjaman (Menunggu, Disetujui, Ditolak)
- Inline update status pada tabel
- Search & Filter data peminjaman
- UI modern menggunakan TailwindCSS
- Reusable components

## Tech Stack

- React
- TypeScript
- TailwindCSS
- React Router DOM
- Fetch API
